Lovebyte Arcade

About this Game

Lovebyte Arcade is a cozy collection of playful, heart-inspired mini games made with charm, creativity, and a touch of whimsy. Each game is designed to feel cute, vibrant, and visually delightful, all while blending soft aesthetics with interactive gameplay that’s just as fun to play as it is to look at.

How to Play

Pick a game from the arcade and jump right in! Follow the instructions shown beside each game as you play. Every mini game is heart-themed, with its own simple rules and playful objectives, just jump in and enjoy discovering how each one works!

Project Breakdown

Project Intent

I created this game hub to deepen my understanding of core front-end development concepts through hands-on building rather than just theory. My goal was to design and develop a collection of small, interactive mini games that explore how structure, styling, and logic come together in real web experiences. Using HTML, CSS, and JavaScript, I focused on turning fundamental concepts into functional, engaging games that feel cohesive as a single arcade-style project.

At the same time, I wanted this project to serve as a portfolio piece that reflects my personality and design style, something visually cute, creative, and expressive that demonstrates both my technical abilities and my interest in building playful, user-focused interfaces.

Tools & Techniques

This project was built using HTML for structure, CSS for layout, styling, animations, and responsive design, and JavaScript for interactivity and game logic. I used event listeners to handle user interactions, DOM manipulation to dynamically update the UI, and conditional logic to control game states such as matching, scoring, and win/loss conditions.

I also incorporated techniques such as class toggling for animations (like card flipping), array manipulation for tracking game progress, and basic state management to control gameplay flow. CSS flexbox and grid were used to structure layouts, with media queries applied for responsive design across different screen sizes.

What I Learned

Through building this project, I strengthened my understanding of how HTML, CSS, and JavaScript work together to create interactive experiences. I gained more confidence in managing game logic, especially tracking state, handling user input, and updating the interface dynamically based on actions.

I also improved my problem-solving skills through debugging and learned how important it is to structure code in a way that remains readable and scalable as complexity increases. Most importantly, I learned how to take abstract front-end concepts and turn them into fully functional, visually engaging projects that feel complete and intentional.
