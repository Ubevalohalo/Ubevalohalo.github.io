Heart & Harvest Matching Game

About this Game

Welcome to Heart & Harvest Matching Game, my very first game!

Each card has a lovely array of hearts as the front face, and you will find that each card, when clicked, reveals adorable pairs of fruits with a consistent heart theme.

I hope you enjoy playing my game as much as I enjoyed developing it! Good luck!

How to Play

Select a card, any card!

Match the fruit of the card with another but beware - if the two fruits selected do not match, you will lose an attempt and inch closer to losing! Match all cards correctly and you will win, if not, game over!

Start by selecting a card.

Project Breakdown

Project Intent and Takeaways

Being my first ever game developed, the main goal within this project was to better familiarize myself with front-end concepts in JavaScript. These main concepts mainly target base level studies, such as arrays, for loops, if else statements, and that in which is encapsulatyed in manipulating them. I also used this project as an opportunity to better my skills in debugging and understanding scope for variables. While not fully immersive within these topics, it has been a great opportunity to be introduced to these fundamental and foundational ideas that will open many doors for learning and understanding in the future.

Tools and languages used include Github, VS Code, HTML5, CSS, and JavaScript.
